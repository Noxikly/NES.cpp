# NES.cpp эмулятор

## Что это?
NES.cpp - простой и легковесный эмулятор с поддержкой модульности мапперов. Поддерживает 0-4 мапперы и работает в 60 fps

## Как установить?
Зависимости для сборки:
* LuaJit
* clang

**Linux/MacOS:**
```bash
git clone https://github.com/Noxikly/NES.cpp
cd NES.cpp
make -j
```

**Windows:**
```
скоро
```
